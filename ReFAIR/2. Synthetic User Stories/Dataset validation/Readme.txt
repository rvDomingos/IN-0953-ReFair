In this folder we released:
1) The random validation subset of user story that we used to construct the dataset validation experiment with the practitioners, rather than a preliminary analysis of the responses;
2) An example of survey questionnaire that contains the overall questions that we asked to the practitioners;
3) A CSV containtained the overall responses of the practitioners (that we used to obtain the dataset validation results);
4) The analysis notebook that contains both the datasets validation scripts and the relative results (please import in the same folder the response csv to correctly run the scripts).
