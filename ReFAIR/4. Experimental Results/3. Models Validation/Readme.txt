In this subfolder we released the detailed results and results of the additional validation experiments 
we conducted with the ML engineering practiitoners. In details this folder contains:

1) A subfolder containg an example of survey we proposed to the practitioners to generate the additional requriements dataset, 
the survey problem scenarios and the practitioners'responses;
2) A subfolder containing the USs story dataset, the testset we formalized collecting the practitioners responses;
3) A copy of the entire ReFair script and the inner models, to perform the model validation analysis;
4) The additional analysis script (as python notebooks) and the spreadsheet containg both the practitioners'USs and results.
5) the "requirements.txt" file to run the ReFair validation script.
