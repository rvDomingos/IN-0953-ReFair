In this subfolder we released all the materials related to the third RQ of our paper (the overall pipeline evaluation), in details the folder contains:
1) a subfolder datasets contains the resources used to validation spreadsheets: the Oracle and a report of the refair predictions related to all the user story of our synthetic dataset;
2) a subfolder prediction contains spreadsheets regarding the ReFair detailed predictions, the oracle and a unified file "overall-predictions.xlsx" required to run the validation scripts;
3) the scripts used to evaluate the RQ3 evaluation metrics (the levenshtein and teh mojo distance), and the detailed analysis of the misclassified output;
4) A spreadsheet that provide the user details about the user story for wich ReFair wrongly classify the application domain with respect to the oracle;
5) A spreadsheet that provide the user details about the user story for wich ReFair wrongly classify the machine learning tasks with respect to the oracle;
6) A copy of the ReFair script that we used to compute the additional validation statistics that we discussed in the appendix technical report;
7) An image that report the additional statistics computed.
