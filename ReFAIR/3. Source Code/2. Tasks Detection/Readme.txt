In this folder we released:
1. The analysis script to identify the best classifier for the Refair's ML task classification (RQ2), that allow to perform directly the training of the 
domain classification models based on the 10-fold cross validation approach and the computation of the relative metrics;
2. A spreadsheet relative to the relations between the low level and high level machine learning techniques (used for the stories generation) and the ML Tasks to classify;
3. A spreadsheet that allow user to observe the training data for the ML tasks classification script, i.e., 
the machine learning tasks associated to each user story of the synthetic dataset that we computed on the relations among the MLTasks to predict and the ML techniques used for the story generation.
4. The requirements.txt file that contains all the dependency of the analysis script;
5. A copy of the synthetic dataset, to run the scripts automatically.
