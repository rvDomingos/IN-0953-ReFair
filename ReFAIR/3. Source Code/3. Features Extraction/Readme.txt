In this folder we released:
1) a notebook that emulate the ReFair's features suggestions process in relation to a given learning task and a given application domain;
2) A subfolder dataset, that contains:
	2.1) A copy of the synthetic dataset;
	2.2) the mapping between application domains and sensitive features (required for the feature extraction process)
 	2.3) the mapping between machine learning tasks and sensitive features (required for the feature extraction process)
